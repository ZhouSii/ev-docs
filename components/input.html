EvInput 输入框 ​

基于 van-filed 二次封装的输入框

用法 ​

显示代码
vue
<template>
  <ev-input label="输入框" v-model="input" placeholder="Please input" />
</template>

<script lang="ts" setup>
import { ref } from 'vue';
const input = ref('');
</script>

提示

ev-input 属性全部支持 van-filed 的属性，以下属性仅展示了部分，具体可查看van-filed官方文档。
